# SINSI

Alerts post automatically. You don't need to do anything.

## Commands

```
/lookup TICKER    On-demand snapshot for any ticker
/add TICKER       Add a ticker to the watchlist
/remove TICKER    Remove a ticker from the watchlist
/list             Show the watchlist
/perf             Show price performance since each ticker was added
/scores           Show squeeze scores for all watchlist tickers
/filter           View and edit discovery filter settings
```

`/filter` opens a panel showing the current settings with three edit buttons:
- **Scoring & Size** — min score, market cap range, max float
- **Price & Buy** — price range, minimum buy value
- **Roles & Discovery** — exec-only or all insiders, enable/disable
